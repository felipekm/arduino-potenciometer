# Arduino Potenciometer

A POC build for the Arduino UNO using [Johnny Five JS](https://github.com/rwaldron/johnny-five)

## Arduino Tools

- 1 Arduino UNO R3;
- 1 USB Cable AB;
- 1 Protoboard
- Male-Male Jumpers
- 1 Resistor 330R;
- 1 Potenciometer 10K;
- 1 Color LED;
